\# 🎤 Acoustic-Based AI Driven Interview System



\## 📌 Overview

The \*\*Acoustic-Based AI Driven Interview System\*\* is an intelligent web application that evaluates candidates based on their spoken responses. It uses audio processing, speech-to-text conversion, and NLP techniques to analyze interview performance and generate feedback.



\---



\## 🚀 Features

\- 🎤 Speech-to-Text using Whisper AI

\- 🎥 Audio extraction from video using FFmpeg

\- 🧠 NLP-based answer evaluation

\- 📄 Resume parsing

\- 📧 Automated email feedback system

\- 💾 SQLite database for storing user data



\---



\## 🛠️ Tech Stack

\- \*\*Frontend:\*\* HTML, CSS

\- \*\*Backend:\*\* Flask (Python)

\- \*\*AI/ML:\*\* Whisper, NLP

\- \*\*Database:\*\* SQLite

\- \*\*Tools:\*\* FFmpeg



\---



\## ⚙️ How It Works

1\. User uploads interview video/audio

2\. Audio is extracted using FFmpeg

3\. Speech is converted into text using Whisper

4\. NLP analyzes the response quality

5\. System evaluates performance

6\. Feedback is stored and sent via email
